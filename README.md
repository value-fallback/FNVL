## Installation

**Via [NPM](https://www.npmjs.com):**

[![NPM](https://nodei.co/npm/fnvl.png?downloads=true&downloadRank=true&stars=true)](https://www.npmjs.com/package/fnvl)

```
npm install fnvl
```

**Via [Yarn](https://yarnpkg.com):**

```
yarn add fnvl
```
